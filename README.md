# 📜 Descrição

Criação da aplicação Dogs no curso de React da Origamid.

# ✨ Tecnologias

- React 
- React Router Dom
- Victory 
- Styled Components

# ℹ️ Como utilizar

    Clone esse repositório
    $ git clone git@github.com:EduardoHoths/Dogs-Origamid.git

    Instalar dependências
    $ npm install

    Inicie o projeto
    $ npm start
